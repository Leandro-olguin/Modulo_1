- El archivo .html se comienza con "\<html>\" y termina con "\</html>\"  

- Para abrir una etiqueta usamos "\<   >\" y para cerrar se utiliza "\</   >\"  

- Para crear el cuerpo se escribe "\<body>\"  

- Para crear un párrafo "\<p>\"

- Para títulos se pueden usar desde "\<h1>\" a "\<h6>\"

- Para una lsita desordenada "\<ul>\"  

- Para un elemento de lista "\<li>\"
